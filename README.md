# Smart Task Manager

A modern and responsive task management application built using Vanilla JavaScript. This app helps users efficiently manage daily tasks with real-time updates and a clean UI.

## 🚀 Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Real-time task filtering (All / Completed / Pending)
- Local storage support (data persists after refresh)
- Dynamic DOM updates
- Responsive design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 💡 Concepts Used

- DOM Manipulation
- Event Handling
- Local Storage API
- Array Methods (filter, map)
- State Management (basic)

## 📌 Future Improvements

- Drag & Drop functionality
- Due dates & reminders
- Dark mode
- Backend integration
